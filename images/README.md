# Images Folder

Please add your profile photo here with the filename: **profile.jpg**

## Instructions:

1. Save your professional photo as `profile.jpg` in this `images` folder
2. Supported formats: JPG, JPEG, PNG, WEBP
3. Recommended size: At least 800x800 pixels for best quality
4. The image will be automatically cropped to a circle and displayed in:
   - Hero section (large circular image)
   - About section (medium circular image)

## Alternative filename:

If your image has a different name or extension, update the `src` attribute in `index.html`:
- Line ~65: `<img src="images/profile.jpg" ...>` (Hero section)
- Line ~82: `<img src="images/profile.jpg" ...>` (About section)
